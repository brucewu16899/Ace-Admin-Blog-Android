#开发背景
写了网页版的[博客](http://www.csi0n.com)怎么能没有客户端？！

#说明
- 使用的是MVP模式开发
- 使用了Rxjava响应式开发,butterknife注解,picasso加载图片,eventbus传递事件,retrofit2+OKHttp网络数据交互
- 同样基于Material Design设计风格
